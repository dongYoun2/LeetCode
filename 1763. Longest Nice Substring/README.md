### Hash Table Optimization


From a pure brute-force approach (or more directly from a problem statement), we can notice that this problem requires a membership check (does pair exist?) for each character in the substring. Always good to be aware that using hash table data structure (set or dictionary in Python) can perform this in $O(1)$ time.

Though this approach is still a brute-force solution in a sense that it checks all possible substrings, properly using the hash table (or set) can reduce the time to $O(n^2)$ time.

[Submission](https://leetcode.com/problems/longest-nice-substring/submissions/1983788090/)—Runtime: 26 ms (beats 33.73%), Memory: 19.32 MB (beats 44.15%)

- TC: $O(n^2)$
- SC: $O(26) \rightarrow O(1)$

```python
class Solution:
    def longestNiceSubstring(self, s: str) -> str:
        n = len(s)
        ans = ""

        for i in range(n):
            seen = set()  # chars in current substring

            for j in range(i, n):
                seen.add(s[j])  # grow substring

                # check "nice"
                ok = True
                for c in seen:
                    if c.swapcase() not in seen:
                        ok = False
                        break

                if ok and (j - i + 1) > len(ans):
                    ans = s[i:j+1]

        return ans

```


## Divide and Conquer Solution

The key idea is that we can find a "bad element" that invalidates the whole substring, and can easily know that that element cannot be part of any nice substring. Then, we can split around that "bad element" (left and right substrings), and recursively find the longest nice substring in each left and right.


[Submission](https://leetcode.com/problems/longest-nice-substring/submissions/1983790346/)—Runtime: 0 ms (beats 100.00%), Memory: 19.45 MB (beats 15.90%)

- TC: $O(n^2)$
- SC: $O(n)$ (recursion stack)

```python
class Solution:
    def longestNiceSubstring(self, s: str) -> str:
        def dfs(sub):
            if len(sub) < 2:
                return ""

            chars = set(sub)

            for i, c in enumerate(sub):
                if c.swapcase() not in chars:
                    # split at bad char
                    left = dfs(sub[:i])
                    right = dfs(sub[i+1:])
                    return left if len(left) >= len(right) else right

            return sub  # whole substring is valid

        return dfs(s)

```

- "exists / membership condition" → use set/hash
- "invalid element breaks everything" → try divide & conquer
