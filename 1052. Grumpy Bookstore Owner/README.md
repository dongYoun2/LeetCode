## Sliding Window Only (Optimal)


[Submission](https://leetcode.com/problems/grumpy-bookstore-owner/submissions/1985466761/)—Runtime: 20 ms (beats 29.36%), Memory: 22.30 MB (beats 7.85%)


- TC: $O(n)$
- SC: $O(1)$

```python
class Solution:
    def maxSatisfied(self, customers: List[int], grumpy: List[int], minutes: int) -> int:
        n = len(customers)
        k = minutes

        # Step 1: already satisfied customers
        base = sum(c for c, g in zip(customers, grumpy) if g == 0)

        # Step 2: sliding window for extra gain
        # gain = customers rescued in first window
        gain = sum(c for c, g in zip(customers[:k], grumpy[:k]) if g == 1)

        max_gain = gain
        for i in range(k, n):
            # remove left
            if grumpy[i-k] == 1:
                gain -= customers[i-k]

            # add right
            if grumpy[i] == 1:
                gain += customers[i]

            max_gain = max(max_gain, gain)

        return base + max_gain
```
